# Passo a Passo para Criar e Iniciar um Projeto Expo

1. **Cria o projeto Expo**  

   npx create-expo-app app_expo_renan


2. **Navegue até a pasta do projeto criado:**
   
    cd app_expo_renan

2. **Inicie o servidor de desenvolvimento do Expo:**

    npx expo start
